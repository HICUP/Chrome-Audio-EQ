Chrome audio EQ

Chrome HTML5 Audio Equalizer

Download .crx Download .tar.gz View on GitHub

With Chrome EQ you can control your audio and video from all tabs in one place.

For best results install HTML5ify. it will force popular sites to use HTML5 instead of flash. (Still lot of work to do...)
